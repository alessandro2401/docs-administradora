## Bem-vindo ao Portal de Documentação

Este portal centraliza toda a documentação dos sites, sistemas e processos da **Administradora Mutual**. Aqui você encontrará guias de uso, referências técnicas, procedimentos operacionais e muito mais.

### Objetivo

Facilitar o acesso à informação, melhorar o onboarding de novos colaboradores e manter a documentação sempre atualizada através de sincronização automática com os repositórios dos sites.

### Estatísticas do Ecossistema

| Indicador | Valor |
|-----------|-------|
| Sites documentados | **23** |
| Projetos no Vercel | **19+** |
| Sites ativos (200 OK) | **20** |
| Sites com redirect válido (307) | **2** |
| Sites em desenvolvimento | **1** |
| Documentos no ecossistema | **22 páginas detalhadas** |

### Sincronização Automática

Este portal utiliza webhooks do GitHub para detectar automaticamente alterações nos sites e atualizar a documentação correspondente. Toda mudança nos repositórios é registrada automaticamente com timestamp e autor.

### Como Usar

1. **Navegue** pelas seções usando o menu superior ou os cards acima
2. **Busque** conteúdo usando a barra de pesquisa (Ctrl+K)
3. **Contribua** reportando erros ou sugerindo melhorias via GitHub
4. **Ecossistema** — acesse a documentação completa de todos os 23 sites na seção [Ecossistema Completo](/ecossistema/)
