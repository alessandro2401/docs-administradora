# Due Diligence

**URL:** [formulario.administradoramutual.com.br](https://formulario.administradoramutual.com.br)

## Visão Geral
Ferramenta de análise de viabilidade para entrada de novas associações ou carteiras no ecossistema da Administradora Mutual. O formulário coleta dados essenciais para uma avaliação de risco e conformidade.

## Dados Coletados
- **Informações Cadastrais:** Dados da entidade e responsáveis legais.
- **Estrutura Operacional:** Volume de associados, ticket médio, abrangência geográfica.
- **Indicadores Financeiros:** Faturamento, sinistralidade histórica.
- **Documentação:** Upload de estatutos, balanços e certidões.

## Processo
1.  **Preenchimento:** O parceiro preenche o formulário online.
2.  **Análise Preliminar:** O sistema realiza validações automáticas.
3.  **Avaliação Técnica:** A equipe de Compliance analisa os dados.
4.  **Parecer:** Emissão de relatório de aprovação ou recusa.

## Acesso
Livre para preenchimento (link público), mas o acesso aos relatórios é restrito.
