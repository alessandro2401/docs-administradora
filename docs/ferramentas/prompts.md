# Gerador de Prompts

**URL:** [prompts.administradoramutual.com.br](https://prompts.administradoramutual.com.br)

## Visão Geral

O **Gerador de Prompts** é uma ferramenta interna da Administradora Mutual, projetada para capacitar colaboradores na criação e otimização de prompts para uso em Inteligências Artificiais (ChatGPT, Claude, Gemini, etc.). O objetivo é padronizar a comunicação, aumentar a eficiência no uso de ferramentas de IA e centralizar o conhecimento da empresa em um único acervo de alta qualidade.

## Acervo de Prompts

Atualmente, a plataforma conta com um acervo de **970 prompts profissionais**, distribuídos em 44 categorias. Este número é atualizado constantemente com novas integrações e sugestões dos próprios colaboradores.

## Funcionalidades Principais

- **Biblioteca de Templates:** Modelos prontos para e-mails, relatórios, análises jurídicas, atendimento, geração de vídeos e imagens.
- **Busca Inteligente:** Filtre por categoria ou pesquise por palavras-chave para encontrar o prompt ideal.
- **Favoritos:** Salve os prompts que você mais utiliza para acesso rápido.
- **Copiar com um Clique:** Botão para copiar o conteúdo completo do prompt para a área de transferência.
- **Sugerir Prompt:** Formulário para que os colaboradores possam enviar suas próprias criações e contribuir com o acervo.

## Guia de Uso

1. **Navegue pelas Categorias:** Explore as 44 categorias na página inicial para encontrar o tema desejado.
2. **Use a Busca:** Digite palavras-chave como "e-mail de cobrança" ou "post para Instagram" para encontrar prompts específicos.
3. **Copie o Prompt:** Ao encontrar o prompt ideal, clique no botão **"Copiar código"**.
4. **Cole na IA:** Cole o conteúdo no ChatGPT, Claude, Gemini ou outra ferramenta de sua preferência.
5. **Personalize:** Substitua as informações entre colchetes `[ ]` pelos dados específicos do seu negócio ou tarefa.
